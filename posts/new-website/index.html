New Website

May 12, 2021 · 1 min read

  • I made this website using Hugo with the Soho theme

  • It took me much longer than I’d like to admit due to something that I couldn’t quite understand from the short instructions that I think might trip up alot of people

  • concept of two repo’s

    • to get the website working you must have two repositories!!!
      Repo function
      code / source the code repo is the default hugo directory made by running hugo new site <name>. However this isn’t what goes in the github pages directory. The public folder generated by hugo -D contains the files that get rendered as your website. you must delete public from this directory! (if you want two repos on github)
      public if you want your code base in a repo you have to add the public folder as a submodule before filling it with hugo generated files. this is the repo that has the <username>.github.io repo title on your github

  • overall I found this process fairly simple I just wish that this distinction was a bit clearer but all in all I am very happy with the results
