Linux Opinion Piece

Aug 2, 2021 · 4 min read

Windows / Linux / BSD / Mac Os

windows

  • I only used windows for a year before I started using linux. I think that windows is probably the worst operating system for consumers / servers / etc… This is harsh but objectively (and in this case subjectively) true. It contains the most back compatible code that leads to 40 year old proprietary code that has yet to be refactored being part of the code base. When the windows XP leak took place we (end users) were able to see bits of this code and just how bad it is. Additionally, this leak exposed bugs / weak points in the codebase ripe for hackers and malicious parties to exploit. This makes Windows the least “safe” operating system. Additionally with programmers in mind window’s only feature that makes it desirable is WSL or windows subsystem for linux, which allows for developers to code in a Linux environment while on windows. Ironically this is really a linux subsystem for windows but to save face “windows” had to come first.
  • Although this review is overwhelmingly negative I respect what Microsoft has been able to do, and especially when they were able to do it. Additionally, Microsoft has been at the helm of most open source projects, essentially uses linux as Azure, and even owns canonical which makes Ubuntu.

Linux

  • I am biased to linux, and maybe I am just a fanboy, but I think that linux is the best OS for almost every programmer and most “consumers.” If you are a programmer no environment comes close to Linux. Every language is a breeze to setup, package managers can help with dependency management, keeps old files in check, and allows for easy upgrades. All of these pro’s carry over to BSD however, what makes linux my favorite is that it really feels like an OS “for the people.” It will run on almost any hardware and can breathe life into old machines. This means that Linux is the most environmentally friendly OS as it reduces E waste. It is a great entrance into the world of computing, command line, and programming for young developers and is slowly gaining traction as a must have skill for most dev jobs and is necessary for all IT jobs.

BSD

  • My only gripe with BSD is the hardware support and support in general. As BSD fans will evangelize BSD is tidy, has a leaner kernel than linux, and has a more involved userbase. This is not false, but when you look at who uses bsd vs linux on average it is clear to see why these differences have come about. BSD is for academics, by academics. When using bsd you have to pick your hardware to fit your OS. Whereas linux is able to fit most hardware. This is the crux of the Linux vs BSD argument for me. If BSD was as hardware compatible as Linux it might be my favorite, but its not, and therefore I can’t recommend it over something that is guaranteed to work now and for years to come on each new generation of CPU and GPU (at least for amd haha).

Mac OS

  • Mac Os was great, but now I wouldn’t suggest most people to use it. With the new icloud scanning that is going on only those completely indifferent to privacy should use it. Mac Os is basically locked down bsd or linux and requires some gymnastics to use in a similar manner. To me this is what I dislike the most about the OS. It makes what should be simple complicated in the hopes of making complicated things simple. It is made for children to be able to use and this paradoxically can make it harder for people that want / need control of their system to use. For the average developer it is not a bad option, and given the new Macbook Air M1 it is probably the best deal for a starting Programmer or CS student however, if you decide to go with Mac OS the recommendations I have would be
    • dont use icloud
    • Use a package manager like homebrew or nix
    • beware of updates as they cannibalize hard drive space with “other” and with every update this will increase and might require a complete reset of the OS to get rid of.
  • In summary mac os is most of the good of Linux with some of the bad from windows all in a shiny, unrepairable but very powerful machine.
